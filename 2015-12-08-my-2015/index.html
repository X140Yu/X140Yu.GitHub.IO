我的 2015

回看一眼今年年初写的这篇博文，真是一点都没有让自己失望，计划一项都没有完成_(:з」∠)_。看来以后还是不要制定计划了，就算制定了，也悄悄地放在心底，不要让别人知道，嗯。

翻译斯坦福公开课

年初的时候，组织并翻译了斯坦福白胡子老爷爷的 iOS 8 开发公开课，翻译项目地址。项目大概从二月初持续到了七月份，期间认识了几个有趣的人，发生了几件有趣的事情，同时也解锁了一些经验和成就。

因为那时候刚开始学习 iOS 不久（也就一两个月），一开始看到公开课的消息的时候我就想着，这门课我学定了。其实全英文的 MOOC 之前我跟过好几个，有的 MOOC 是会有字幕组翻译来造福大众的。由于这门课很新，还没有人开始翻译。我就想，既然有能力承担一些事情，还可以在这期间充实丰富自己，那为什么不去做呢？想法有了，那就去实现吧。

我下载了第一集视频，看了一遍，在 GitHub 上建了个项目，于是就开始翻译了。一开始的心情无疑是兴奋的，对着视频，坐着几个小时，翻译了几百行，都不觉得累。可一个人的时间和经历是十分有限的，如果我一个人把所有的业余时间都放到这上面，估计也得很久很久才能翻译完成，这样不够高效。于是我萌生了与其他人一起翻译的想法，而现在传播这些消息最有效的方式就是发微博并 @ 一些大V，我真就这么做了，也收到了很好的效果。微博的曝光量非常高，很多人加入 QQ 群，说想要翻译任务。

当时的 iTunes 视频里是有字幕的，但是我试过了几个字幕提取软件都不能把其中的内容提取出来，于是就傻呵呵地以为字幕真就是提取不出来的。字幕翻译嘛，怎么可能不带英文字幕，然后我就和几个人一起做人肉的字幕提取工作。事实证明这么做是非常 2 的，群里的一位同学提醒说 Windows 平台的 CCExtractor 可以很容易地把字幕提出出来。干！我怎么没有想到 Windows 平台。这算一个小插曲吧，在这之后，翻译工作就有条不紊地进行下去了。

在项目进行过程当中，我很高兴认识了周奕飞（AustinChou0126)。他在一开始的时候就跟我一起管理整个项目，一直到项目的最后。从他那里我了解到了很多，也学到了很多。恰好他也在哈尔滨读书（HIT），后来通过线下观看 WWDC 2015 直播的机会正式面基 XD。

其实整个项目结束下来，带给我个人的，不光是在英语翻译这个层面的提升，还有对于任务的分配，工具选择重要性的理解，更重要的是在一件事情上，有始有终，坚持到底。说实话，后期的翻译任务很重，愿意翻译的人员越来越少，之前一集交给三五个人，但是到最后，都是一人翻译一集，但还有的集数没有人认领。有段阶段自己都懒得不行了，不过还好一路坚持了下来。这个项目 截止今天一共有了 2000 多个 star，还被网易公开课给采用了。应该可以帮到很多英语一般但想要学习 iOS 开发的同学们吧，这样就好了 :]

书趴网

大概四五月份，在网上无意中看到了隔壁黑龙江大学的一个创业项目 - 书趴网。了解了情况以后，我就决定跟他们一起做出点东西来（到今天我也没做个成品出来实在惭愧 = =#）。在这期间结识了几个小伙伴，林浩，谭岑，春潮。虽然有的比我还小，但是都是很有想法的。跟他们在一起我也学到了不少知识。

理工助手

这可能是我独立完成的一个最完整的 project 了，从设计（如果那也算设计）到开发再到上架。如果想了解那段往事，可以读一读这篇博文。

知乎

去知乎实习是在我计划之外的，其实我本来想在九月份的校园招聘大展身手的（强行不要脸），没想到知乎的实习生面试通过了。那是我第一次面试通过，接到 offer 后，心情异常激动，只跟一个老师打了招呼就放弃了学校的所有课程，前往帝都。

我不想过多描述在帝都的居住条件，毕竟是去实习，没想太多，就租了一个小隔断。你懂的，真的很糟糕。那是我一生中住得最差的地方，没有之一。

好啦，不谈那些煽情的了，好好谈谈知乎的实习经历吧。

首先是工作环境。好，真的好。工程学座椅（我的最爱）+ DELL 外接显示器 + 早午餐 + 无限零食和饮料。公司的活动也特别多。一到公司就赶上了一年一度的 Outing，到崇礼，骑行 + 穿越。中秋节的时候，大家还一起做了月饼。万圣节的时候大家还会穿着奇装异服，那天刚好是我和 CEO 周源的转正面试，，他 cos 成了四代目，非常有趣。总之公司对员工的关怀非常到位。之前几天北京发布了雾霾红色预警，公司还可以让大家在家里办公，真的是太 nice 了有木有。

再来谈谈我的收获。工作方面，在来公司之前，没做过 Code Review，所以一开始提交的代码并不是太符合预期。有一些 BUG，虽然可以改好，但是改法太 trick，不便于后期维护。后来渐渐好了，对于问题能够分析出它的原因所在，也开始逐渐适应工作。对于交付的工作大多都能及时完成，有困难的地方也会去跟同事们交流，一起寻找解决的方案。在设计方面有疑惑的地方，会和设计师进行沟通，避免自己做的东西出现问题，或者做错了方向，成了无用功。

在实习期间，我的感受有以下几点：

  1. 与同事之间的沟通是很重要的。
  2. 写代码不可以着急，上键盘之前先在脑袋里构思一下。
  3. Push 代码之前，要自己 check 一下，避免同事的重复劳动。

到了今天，我的三个月（两个月 + 一个月的 remote）的知乎实习生涯正式结束了。其实我可以选择继续做下去直到毕业的，但是我还是想歇息一段时间。因为自己心里还有些小算盘，想在毕业之前完成一些自己真正想做的事情。实习工作固然能收获开发经验和一些收入，但是毕业之后，就再也没有现在一样想做什么就能去做什么的大块时间了。我相信我自己的选择。

最后再来谈谈帝都吧，其实除了雾霾和堵车，这个城市给我的印象非常好。具体是怎么好呢，就是在那里的时候不觉得她怎么地，但是离开那里以后，特别地想念。不过没关系啦，明年还是会去的(╯3╰)

总结

其实 2015 年对于我的意义还是很重大的，在这一年里我完成了部分从校园生活到真实工作之间的转变。这一年我拥有了很多，也失去了很多。失去了的可能永远都不会再回来了，可是经历过的都会烙印在我的心底永远不会磨灭。

明年我就 22 岁（周岁，显小）了，快到新的一年了，祝自己在下一年里，一切顺利吧~
