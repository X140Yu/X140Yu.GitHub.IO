一路走来，我经历过许多寒假。今年的这个寒假应该算是我学生时代寒假的终结吧。不同于以往寒假的「不知道该做点什么」，这最后一个寒假我已经知道点自己该做什么，并付出实践了。

学习开车

在这两个月，我花了最多时间做的一件事情可能就是学习开车了吧。

据我的完全统计以及估算，我花费在学车上的时间至少在 8（科目二次数） * 12（小时） + 5（科目三次数） * 5（小时） = 121（小时）以上。学过车的同学们都懂，学车的过程其实没有那么轻松，在这里我就不具体阐述了，反正能遇到一个好教练就嫁了吧。

我建议所有同学在有时间和经历的情况下都学习一下如何开车。因为在学车期间，你可能会为了练车而养成早睡早起的好习惯；你也可能会试着接触陌生人，去和他们交谈，因为只属于你自己一个人练车的时间几乎没有；你可能会为了学倒车入库，查了很多教程，脑补了许多种情况；你可能会喜欢坐在副驾驶，观摩老司机是如何开车的；你的脸皮可能会被磨练地越来越厚，不管以后别人说你什么，内心都会十分平静；你可能会因此爱上开车，因为放着自己喜欢的音乐，在不堵车的道路上奔驰，会是一件十分美妙的事情。

总之我能在学车这件事上传授给你们的经验有：关于驾校和教练的选择，最好是经过别人介绍，千万不要自己随便找个驾校去冒险，这样体验不好的机率会很高；一切听教练的，不要自创一些招式，因为通过考试和上路开车是两个概念；能去练就要练，千万不要懒。

学习五笔

我还利用这个寒假时间学会了如何使用五笔输入法。

从这一个多月的时间看来，这也是一件会对我产生深远影响的事情。随着我花在电脑上的时间越来越多，我接触书面写字的机会越来越少了，提笔忘字是一个很严重的问题，我不想搞到最后连字都不会写了，于是就学习了一下五笔。像我这么酷炫的人，这当然不是全部的理由。我总觉得会五笔的人逼格都很高（其实我很喜欢 zhuang bi），于是就开始学了啊。其实去年自己也学过一些，但是练了一半就放弃了，因为跟我飞快的拼音输入速度相比，这简直慢爆了啊！我没能熬过开头最痛苦的那段时期，遂弃之。上个月可能因为自己没有什么要紧的事情，还是把五笔给捡起来了。

学习的过程其实不那么痛苦，字根什么的可以不先背，而是边练边记。我最初是通过金山打字通上手的，跟着里面的教程循序渐进地练习，等到常用字拆得差不多了，我就把所有输入法都换成了五笔无论电脑还是手机。开始不适应是肯定的，字不会写，笔顺错误，不会拆字，等等，会出现一系列的问题。可是这个过程必须要有，学习五笔并不需要很聪明的脑袋，而是需要勤奋地练习。

等到真正训练成了以后，你会发现，用五笔录入文字是一件很愉快的事情：三四个字母就能完全确定一个字，基本不会有不停 +/- 翻页然后选字的过程；你会发现很多字的笔顺并不是你想像的那样，比如「可」是先输入「丁」然后是「口」，「力」是先「丿」后「乛」，诶？还凑巧学会了如何打偏旁这么高逼格的事情；我还纠正了自己的许多错别字，比如是「厉害」不是「利害」等等；你会在走路或者坐车的时候望向街边的招牌，然后想着这个字该怎么拆；你会在和别人聊天的时候更加温柔，因为相比于拼音「说」的感觉，五笔更有一种「书写」的感觉…这东西还是得各位看官自行体会，总之相比于过程的痛苦，结果是很赞就是了。

一些小事

使用豆瓣

使用豆瓣标记自己读过什么书，看过什么剧，真的是一个很好的习惯。可以在看一个东西之前上去看看评分和相关的评价，不能说完全靠谱，但可信度还是比较高的。当你看到自己标记读过的书少的可怜的时候（比如我），可能会激发起一点点斗志吧 XD

使用吾志

吾志网是我无意中发现的一个可以记录自己小想法，写点小日记的网站。这个网站很简单，只能记录文字，而且不可以删除。我在想发状态的时候，可能会先写到吾志上，等那个劲儿过去了，我就不想发状态了。这也部分地治疗了我的社交瘾。

Ingress

玩 Ingress 完全是某一天的无聊之举，那时的我可能不知道这游戏是怎么一回事儿。可是在我真正了解并上手了以后，就再也放不下手机了。它不像我通常看到的大部分游戏。官方称它为现实增强类型的游戏，而对我来说，它对我产生的直接影响就是，我越来越不像个宅男了！我越来越喜欢走出家门了。这也属于「玩过的人都懂」那个范畴。

学习 Rails

Rails 是一个 Web 的后端框架，以敏捷开发和编程语言（Ruby）友好而出名。因为毕业设计是网站相关，所以我就花了些时间了解并学习 Rails。但学习的效果一直不明显，还是云里雾里的，可能跟我不熟悉 Web 有很大关系。还是需要多花些时间好好学学 Rails 才可以，因为我真的很喜欢（不了解也敢谈喜欢 →_→）。

赚点外快

寒假还给国外某所大学 CS 专业的同学们（没看错，是好几个）写过 Python 和 C 专业课的作业，还给朋友的朋友搭了个网站，算是赚了一些零花钱。虽然没那么多，但也算是对我程序员身份的一种肯定吧。

过去的寒假，我要么是在发呆，要么是在去发呆的路上。今年的寒假能过成这样，也算是一种成长吧。
